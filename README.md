# GB_Emulator

This is a simple GameBoy emulator running in C.
currently it can run Tetris and Dr Mario.

## Controls
  * Movement -Arrow keys
  * Select - Space
  * Start - Enter
  * A,B - A,S
  * Exit - Esc

## Build Instructions
  * browse to directory
  * run make

## Run Emulator
  ./Out/gb_emu PATH_TO_ROM
